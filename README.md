# Excel O'qitish Rejasi — Interactive Learning Roadmap

Noldan professional darajagacha mo'ljallangan, 5 bosqichli interaktiv Excel o'quv dasturi platformasi.

## 🚀 Texnologiyalar
* **React 18** — UI komponentlar
* **Tailwind CSS** — Zamonaviy va moslashuvchan (responsive) dizayn
* **CoreX Design System** — To'q yashil (Dark-green) premium interfeys va vizual progress tizimi

## 📊 Kurs ko'rsatkichlari
* **5** Bosqich
* **7** Hafta
* **28+** Darslar
* **100%** Amaliy mashg'ulotlar

---

## 🗺️ Kurs Strukturasi (Roadmap)

### 🟢 1-Bosqich — Excel bilan tanishuv (1-2 hafta)
* Interfeys, katak, qator va ustunlar bilan ishlash.
* Ma'lumotlarni to'g'ri kiritish va tahrirlash.

### 🟡 2-Bosqich — Oddiy formulalar (2 hafta)
* Arifmetik amallar.
* Asosiy funksiyalar: `SUM`, `AVERAGE`, `MAX`, `MIN`, `COUNT`.

### 🟠 3-Bosqich — IF va mantiqiy formulalar (1-2 hafta)
* Shartli hisob-kitoblar.
* Mantiqiy operatorlar: `IF`, `AND`, `OR`.

### 🔵 4-Bosqich — Formatlash va dizayn (1 hafta)
* Ranglar, chegaralar va vizual iyerarxiya.
* Chiroyli jadvallar yaratish va chop etishga tayyorlash (Print setup).

### 🟣 5-Bosqich — Diagramma va yakuniy loyiha (1 hafta)
* Grafik va diagrammalar bilan vizualizatsiya.
* To'liq amaliy yakuniy loyiha (Portfolio tayyorlash).
